navToSelect.js

navToSelect is jQuery plugin used to convert your website navigation into a select drop-down menu for small screen devices.

Usage

1. HTML structure

<ul id="nav">
<li><a href="homepage.html">Homepage</a></li>
<li><a href="about.html" class="active">About us</a></li>
<li><a href="contact.html">Contact</a></li>
</ul>

2. CSS

/* default style */
.navToSelect { display: none; }
/* Mobile device */
@media only screen and (max-width: 767px) {
  #nav { display: none; }
  .navToSelect { display: block; }
}

3. JavaScript

/* default style */
<script src="js/jquery.min.js"></script>
<script src="js/navToSelect.js"></script>

Examles

Example 1

  • Home
  • Categories
    • Category 1
    • Category 2
  • Page
    • Page 1
    • Page 2
  • External
  • Go to top

HTML:

<nav id="example-1">
    <ul>
        <li><a href="index.html">Home</a></li>
        <li>
            <a href="cats.html">Category</a>
            <ul>
                <li><a href="cat-1.html">Category 1</a></li>
                <li><a href="cat-1.html">Category 1</a></li>
            </ul>
        </li>
        <li>
            <a href="#">Page</a>
            <ul>
                <li><a href="page-1.html">Page 1</a></li>
                <li><a href="page-1.html">Page 1</a></li>
            </ul>
        </li>
        <li><a href="http://google.com">External</a></li>
        <li><a href="#top">Go to top</a></li>
    </ul>
</nav>

Javascript:

jQuery(function(){
    $('#example-1 > ul').navToSelect();
});
